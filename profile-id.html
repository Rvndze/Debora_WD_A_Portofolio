Learn About Me!

Saya Debora Lisda Graciella dari Program 1 Tahun Komputer Aplikasi Bisnis, STMIK LIKMI. Saya memiliki sebuah hobi yang sudah saya kembangkan sejak tahun 2018, yakni menggambar. Saya mulai terinspirasi untuk belajar menggambar dari novel karya Tere Liye berjudul "Bumi". Saya menemukan banyak fan-art dari novel tersebut, yang menjadi motivasi saya untuk belajar menggambar.

Pada awalnya, saya menikmati proses menggambar secara tradisional (dengan kertas dan pensil). Ketika pandemi COVID-19 menyerang saya mulai mencoba memanfaatkan gadget sebagai media menggambar secara digital. Saya belajar secara autodidak, mencoba beberapa software grafis seperti Ibis Paint X, Medibang Paint Pro, Paint Tool Sai, Adobe Photoshop, dan lainnya. Ke depannya, saya ingin belajar membuat animasi, model Live2D dan Model 3D.
